# Naqiy — Prompts Veo & Nano Banana (43 vidéos)

> Chaque vidéo = 1 prompt visuel (scène IA) + 1 overlay texte (post-prod) + 1 caption Instagram.
> Format : 9:16 vertical (1080x1920) · 10-15s · Dark premium.
>
> **Palette Naqiy** : fond #0C0C0C, or #D4AF37, halal vert #22C55E, haram rouge #EF4444, douteux ambre #F59E0B.
> **Ton** : Calme, Factuel, Inclusif, Humble, Constructif, Direct.

---

## Convention de prompt

**Veo** : prompt descriptif cinématique en anglais, axé ambiance/lumière/mouvement.
**Nano Banana** : prompt en français, plus court, orienté composition visuelle.

Chaque prompt décrit la **scène visuelle de fond**. Le texte (titre, rulings, sources) sera ajouté en post-production (CapCut, Premiere, After Effects) avec la typographie Naqiy (Inter, Noto Sans Arabic).

# NOTES DE POST-PRODUCTION

## Workflow recommandé

1. **Générer** chaque scène visuelle avec Veo ou Nano Banana
2. **Importer** dans CapCut Pro / Premiere Pro / After Effects
3. **Ajouter** les overlays texte avec la typographie Naqiy :
   - Titres : Inter Black, #D4AF37 (or)
   - Rulings : Inter SemiBold, couleur par statut (#22C55E halal, #EF4444 haram, #F59E0B douteux)
   - Sources : Inter Regular, rgba(255,255,255,0.4)
   - Arabe : Noto Sans Arabic, direction RTL
   - Footer : Inter Regular, rgba(255,255,255,0.35)
4. **Ajouter** le watermark logo Naqiy (coin inférieur droit, 8% opacité)
5. **Audio** : ambiance subtile (wind-ambient ou silence) — pas de musique agressive
6. **Exporter** en 1080x1920, H.264, 30fps

| Element | Hex | Usage |
|---------|-----|-------|
| Fond | #0C0C0C | Background des text overlays |
| Or | #D4AF37 | Titres, accents, logo |
| Halal | #22C55E | Texte ruling halal |

| Douteux | #F59E0B | Texte ruling douteux |
| Texte | #FFFFFF60 | Corps de texte, sources |
| Texte dim | #FFFFFF25 | Footer, metadata |
