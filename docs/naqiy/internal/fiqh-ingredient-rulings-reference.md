## 1. PORC (خنزير)

### Versets coraniques (4 versets)
- **Al-Baqarah 2:173** : « إنما حرم عليكم الميتة والدم ولحم الخنزير »
- **Al-Ma'idah 5:3** : « حرمت عليكم الميتة والدم ولحم الخنزير »
- **Al-An'am 6:145** : « أو لحم خنزير فإنه رجس » — le pronom « hu » vise l'animal entier
- **An-Nahl 16:115** : Répétition de 2:173

### Consensus
- **Ijma' absolu** sur l'interdiction de TOUTES les parties : viande, graisse, os, peau, moelle
- Ibn Hazm (al-Muhalla) : liste exhaustive
- Al-Qurtubi : rapport d'ijma' communautaire
- Ibn Baz (Fatwa 19486) : « shahm » inclus dans « lahm »

### Hadith
- Bukhari 2236 / Muslim 1581a (Jabir) : interdiction de vente du khamr, mayta, khinzir, idoles

### Statut : **HARAM** (unanime, 0.99 confiance)

## 2. GÉLATINE (جيلاتين)

### Cas 1 : Gélatine porcine
| Madhab | Ruling | Base |
|--------|--------|------|
| Hanafi | Halal (istihala) | Transformation chimique complète |
| Chafiite | **Haram** | Istihala restreinte; porc = najis al-'ayn |
| Malikite | Halal (istihala, position mu'tamad) | Transformation = purification |
| Hanbalite (majorité) | **Haram** | Transformation incomplète |
| Hanbalite (Ibn Taymiyyah) | Halal | Accepte istihala large |
| Comité permanent | **Haram** | Position hanbalite majoritaire |
| Dar al-Ifta Égypte | Halal | Istihala vérifiée |
| IIFA | **Différé** (Résolution 210) | En étude |

**Sources** : IslamQA #219137, IslamWeb #86671, Ibn Baz #19486, Dar al-Ifta #6891, IIFA Résolution 210

### Cas 2 : Gélatine bovine (abattage non-rituel)
- Hanafi/Malikite : halal (istihala + Ahl al-Kitab)
- Chafiite : haram
- Hanbalite : disputé (Ibn Uthaymeen : halal si pas de traces)
- **IslamWeb #275287** : halal via istihala

### Cas 3 : Gélatine halal-certifiée → **HALAL unanime** (0.98)

### Cas 4 : Gélatine de poisson → **HALAL unanime** (0.97)
- Coran 5:96, Abu Dawud/Ibn Majah (deux morts licites : poisson et sauterelle)

### Cas 5 : Gélatine en médicaments
- Alternative halal existe → haram d'utiliser porcine
- Pas d'alternative + nécessité → daroura s'applique
- Capsule jetable → jeter la capsule, prendre le médicament dedans

## 3. ALCOOL DANS LES ALIMENTS

### Versets coraniques (3 étapes progressives)
1. **Al-Baqarah 2:219** : « plus de mal que de bien » (désapprobation)
2. **An-Nisa 4:43** : « ne priez pas en état d'ébriété » (restriction)
3. **Al-Ma'idah 5:90-91** : « رجس من عمل الشيطان فاجتنبوه » (interdiction finale)

### Hadith clés
- Muslim 2003a : « كل مسكر خمر وكل مسكر حرام » (tout enivrant est khamr)
- Abu Dawud 3681 : « ما أسكر كثيره فقليله حرام » (petite quantité aussi haram)
- Muslim 2361 : « الخمر من هاتين الشجرتين » (palmier et vigne)
- Tirmidhi 1295 : malédiction sur 10 personnes liées au khamr

### Cas 1 : Éthanol comme vecteur d'arôme (<2%)
- **Majorité contemporaine (Ibn Uthaymeen, Ibn Baz, Dar al-Ifta, AMJA, IIFA)** : licite si non-enivrant
- Distinction clé : absorber ≠ ajouter délibérément

### Cas 2 : Alcool naturel (pain, fruits, sauce soja, <0.5%)
- **Halal** quasi-unanime. IIFA Résolution 225 : « pas d'objection aux traces d'éthanol résultant de fermentation naturelle »

### Cas 3 : Cuisson au vin
- **Haram** quasi-unanime. IIFA Résolution 225 : « pas permis d'utiliser l'alcool en cuisine, même une seule goutte »
- L'évaporation ≠ istihala; 5-85% d'alcool subsiste (étude USDA)

### Cas 4 : Alcool en médicament
- Haram par défaut, licite sous daroura (nécessité médicale + pas d'alternative)

### Cas 5 : Bière/vin « sans alcool » (0.0-0.5%)
- **Licite** selon majorité (Ibn Uthaymeen, Comité permanent, IslamWeb)
- Tashabbuh (imitation) déconseillé mais pas haram

### La nuance hanafite
- Abu Hanifa : seul le vin de raisin/datte/raisin sec est khamr; autres enivrants permis en petite quantité
- **Position de fatwa (mu'tamad) : suit Muhammad al-Shaybani** → tout enivrant est khamr, tout est haram

## 4. CARMIN / COCHENILLE (E120)

### Question fondamentale : les insectes sont-ils halal ?

| Madhab | Insectes | Base |
|--------|----------|------|
| Hanafi | **Haram** (sauf sauterelles) | Khabaith (Coran 7:157) |
| Chafiite | **Haram** (sauf sauterelles) | Khabaith + traces pardonnées |
| Hanbalite | **Haram** (sauf sauterelles) | Hasharat interdits |
| Malikite | **Halal** (avec conditions) | Coran 6:145 : seules 4 choses interdites |

### E120 spécifiquement

**Institutions favorables** :
- IslamQA #382570, #162658 : licite via istihala
- Dar al-Ifta Égypte : licite (transformation chimique)
- Iftaa' Jordanie #3240 : licite sous 4 conditions
- MUI Indonésie (Fatwa 33/2011) : halal (cochenille ≈ sauterelle)

**Institutions strictes** :
- Darul Ifta Birmingham (Hanafi) : haram pour consommation
- Certains Hanbalites : pas d'istihala complète

### Statut Naqiy : **DOUTEUX** (ikhtilaf légitime, Malikites = halal)

## 5. PRÉSURE (أنفحة) ET LACTOSÉRUM

### Preuve textuelle majeure
- **Hadith de Tabuk** (Mishkat 4227) : le Prophète ﷺ a mangé du fromage en territoire byzantin
- **Athar de 'Umar** (Ibn Abi Shaybah) : les Sahabas mangeaient le fromage des Zoroastriens
- **Hadith de Salman** (Tirmidhi/Ibn Majah) : « ce sur quoi Allah s'est tu est pardonné »

### Présure animale (abattage non-rituel)

| Position | Ruling | Tenants |
|----------|--------|---------|
| Halal | Abu Hanifa, Ahmad (une narration), Ibn Taymiyyah, Comité permanent (Ibn Baz), IslamQA #115306, Dar al-Ifta #9411 |
| Haram | Majorité Malikite/Chafiite/Hanbalite (mu'tamad) |

**Porcine** : unanimement haram (IslamWeb #47734)

### Lactosérum
- Suit le statut du fromage source
- Présure microbienne → halal sans ambiguïté
- Présure animale → suit les mêmes divergences que la présure

## 6. E471 (Mono- et Diglycérides)

### Source inconnue
- **IslamWeb #387325** + ECFR : licite par défaut (al-asl al-ibahah)
- **IslamQA #114129** : prudence en pays non-musulman
- **Comité permanent** : licite sauf preuve de lard

### Source porcine confirmée
| Madhab | Ruling | Istihala ? |
|--------|--------|-----------|
| Hanafi | Potentiellement halal | Oui (large) mais débat |
| Malikite | Potentiellement halal | Oui (mu'tamad) |
| Chafiite | **Haram** | Non (restreinte) |
| Hanbalite (majorité) | **Haram** | Non |

### Lard / Saindoux
- **Haram unanime** — toutes les écoles, tous les savants
- Al-Qurtubi : ijma' sur interdiction de la graisse de porc
- Ibn Baz Fatwa 19486 : catégorique

### Graisses non-porcines
- Canard/oie : halal si abattage rituel, douteux sinon (mayta)
- Suif de boeuf : halal si abattage rituel, débat Ahl al-Kitab sinon

## 7. VINAIGRE DE VIN

### Cas distingués
1. **Transformation naturelle** → Halal (consensus)
2. **Transformation délibérée par musulman** → Haram (majorité) / Halal (Hanafi)
3. **Transformation par non-musulman** → Halal (Ibn Uthaymeen) / Douteux (strict)

### Sources
- Muslim 2051 (« quel bon condiment ! »)
- Abu Dawud 3675 (interdiction de transformer délibérément)
- IslamQA #276185, #191176
- IslamWeb #81657, #204487
