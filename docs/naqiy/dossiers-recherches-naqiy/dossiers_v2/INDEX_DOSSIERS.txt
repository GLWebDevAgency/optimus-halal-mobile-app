Phase 2 : DONNÉES SCIENTIFIQUES
───────────────────────────────
• Biologie source organism (Kerria lacca, Dactylopius coccus)
• Composition chimique complète
• Processus industriels (extraction, purification, solvants)
• Statistiques production mondiale
• Usages alimentaires, pharmaceutiques, cosmétiques

Phase 3 : ANALYSE 4 MADHĀHIB ÉQUI-PROFONDE
───────────────────────────────────────────
Chaque madhab reçoit:
  • ≥4 textes classiques avec auteur + date mort (d.)
  • ≥3 savants contemporains nommés
  • Position sur l'enjeu fiqhi central
  • Citation arabe originale + traduction fidèle
  • NaqiyScore par école (0-100)

Madhabs : Hanafi, Shafi'i, Hanbali, Maliki

Phase 4 : POSITION NAQIY (IJMĀʿ + PRUDENCE)
────────────────────────────────────────────
• Identification zones ijmā' (consensus)
• Identification zones khilāf (désaccord majeur)
• Verdict interne vs externe distinction
• Principe « Délaisse le douteux pour le certain »
• Recommandations géographiques

Phase 5 : NAQIYSCORE MATRICIEL
──────────────────────────────
≥9 scénarios d'usage:
  • Interne certifié MUI vs sans certification
  • Externe cosmétique vs alimentation
  • Résidu éthanol >0.5% vs pur
  • Région indonésienne vs UK
  • etc.

Chaque scénario: Score (0-100) + Verdict + Rationale

Phase 6 : ANTI-HALLUCINATION STRICT
──────────────────────────────────
• Protocol vérification à 3 passages
• WebFetch confirmation chaque URL
• Triangulation sources classiques (≥2 références)
• Divulgation complète éléments non vérifiés
• Pas d'invention — 100% citabilité
