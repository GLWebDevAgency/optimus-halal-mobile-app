## 1. Identification de la substance

| Paramètre | Valeur |
|---|---|
| **Nom français** | Gomme-laque / Shellac / Vernis confiseur |
| **Nom anglais** | Shellac / Lac resin / Confectioner's glaze |
| **Nom arabe** | صمغ اللك / الشيلاك / إفرازات حشرة اللك |
| **Numéro E** | E904 |
| **Numéro INS** | 904 |
| **Code CAS** | 9000-59-3 |
| **Formule chimique** | Résine polyester complexe (aleuritic acid ~20%, shellolic acid ~5%, jalaric acid esters ~10%, résines ~50%) |
| **Source biologique** | *Kerria lacca* (fem. insecte Hemiptera: Coccoidea) |
| **Origines principales** | Inde (70-80%), Thaïlande (10-15%), Birmanie (5-10%) |
| **Production annuelle mondiale** | ~20 000 tonnes |
| **Rendement** | 50,000-200,000 insectes par kg de shellac raffiné |

### Description technique

La gomme-laque (E904) est une résine naturelle sécrétée par la femelle de l'insecte *Kerria lacca* (anciennement *Laccifer lacca*). Cette sécrétion recouvre des arbres hôtes (Butea monosperma, Schleichera oleosa, Ziziphus mauritiana) où l'insecte se nourrit de la sève. La résine brute (stick lac) est transformée industriellement par étapes successives :

1. **Récolte** : Stick lac (branche couverte de sécrétion + insectes)
2. **Débranchage** : Seed lac (sécrétion isolée, 5-30% débris d'insectes)
3. **Blanchiment/Filtration** : Bleached shellac (solvent-refined, <10% résidu d'insectes)
4. **Dewaxing** : Dewaxed shellac (filtration supplémentaire)
5. **Solvant industriel** : Dilution à l'éthanol pour application alimentaire/pharmaceutique
6. **Finition** : Poudre ou pellets E904

## 2. L'enjeu islamique central : les insectes et leurs sécrétions

### 2.1 Deux questions fondamentales

1. **Les insectes sont-ils *khabā'ith* (choses impures/interdites) ?** Le Coran 7:157 mentionne qu'Allah « rend licites les choses bonnes (al-ṭayyibāt) et interdit les choses impures (al-khabā'ith) ». Les insectes terrestres sont-ils dans la catégorie des choses impures ?

2. **Les sécrétions d'insecte (comme le shellac) héritent-elles de l'interdiction des insectes ?** Ou constituent-elles des substances indépendantes (tahir/pures) ?

### 2.2 Position des écoles (madhāhib)

| Madhab | Position classique | Exemple |
|---|---|---|
| **Hanafi** | Insectes haram (sauf sauterelle) | Pas de consumption possible |
| **Shafi'i** | Insectes haram (sauf sauterelle) | Pas de consumption possible |
| **Hanbali** | Insectes haram (majorité) | Pas de consumption possible |
| **Maliki** | Insectes halal si abattus avec niyyah + tasmiyah | Conditions industriellement impossibles |

### 2.3 Le problème du shellac industriel

Même le madhhab Maliki (le plus permissif) exige que les insectes soient abattus avec **intention rituelle (niyyah)** et **invocation (tasmiyah/bismillah)**. La production industrielle du shellac implique :
- Extraction massive sans intention rituelle
- Séchage à la vapeur/chaleur sans bismillah
- Filtration et blanchiment

**Conséquence logique :** Aucun madhab ne peut justifier le shellac industriel selon les principes classiques.

- **URL canonique :** https://islamqa.info/en/answers/382570
- **Date :** VERIFIED_LIVE (2026-04-07)
- **Verdict :** HALAL
- **Principe islamique :** Istihalah (transformation chimique)
- **Raisonnement direct :** « There is nothing wrong with eating fruits coated with shellac wax or in which carmine has been used, if it does not cause any harm. »
- **Justification théorique :** Le shellac subit une transformation chimique complète changeant sa nature juridique. Principe d'istihlaak (absorption) : les petites quantités de substances interdites absorbées dans des matériaux purs sont overlookées.

## 4. Analyse par madhab

### **HANAFI — POSITION DIVISÉE**

#### Classique : HARAM
- **Sources classiques :**
  - **Al-Hidaya** (al-Marghīnānī, d. 593/1197) : Insectes terrestres parmi les choses répugnantes, non permissibles
  - **Badā'iʿ al-Ṣanāʿ** (al-Kāsānī, d. 587/1191) : Vente d'insectes invalide car utilisation nuisible
  - **Radd al-Muḥtār** (Ibn ʿĀbidīn, d. 1252/1836) : Confirmatio Hanafi insect restriction
  - **Al-Mabsūṭ** (al-Sarakhsī, d. 490/1096) : Détail des insectes impurs

#### Position classique en détail :
Les insectes sans sang qui coule (dam masfūḥ) restent *khabāith* (choses impures) — ne peuvent pas être consommés. L'extraction industrielle ne constitue pas une *istihalah* valide aux yeux de la plupart des juristes Hanafites classiques.

#### Contemporain : DIVISÉ
- **Majorité restrictive :** Darul Ifta Birmingham, DUA Azaadville, écoles traditionnelles
- **Minorité permissive :** Mufti Taqi Usmani (Darul Uloom Karachi), Seekers Guidance Hanafi, positions modernes

#### Position permissive moderne (Mufti Taqi Usmani) :
Le shellac est une sécrétion de glande exocrine (non ingesta de l'insecte lui-même). Les sécretions des glandes exocrines sont classiquement pures (tahir), comme la cire d'abeille et la soie. L'extraction industrielle complète + filtration + blanchiment = istihalah satisfaisante.

#### NaqiyScore Hanafi : 45/100 (MASHBOOH)

### **SHĀFI'Ī — HARAM (majorité)**

#### Position : HARAM
- **Sources classiques :**
  - **Al-Umm** (Al-Shāfiʿī, d. 204/820) : Interdiction des insectes parmi les khabā'ith

  - **Tuḥfat al-Muḥtāj** (Ibn Ḥajar al-Haytamī, d. 974/1566) : Confirmation Shafi'i

#### Raisonnement :
L'insecte Kerria lacca est classé parmi les hasharat (insectes terrestres) → khabā'ith (choses impures/répugnantes) → interdiction coranique Qur'an 7:157. Le shellac hérite directement de cette interdiction.

#### Exception :
Les sauterelles/locusts (جراد/jarad) sont permises par consensus (ijma') de tous les madhāhib car preuve textuelle explicite.

#### Condition Maliki critique : NIYYAH + TASMIYAH
Pour qu'un insecte soit halal Maliki, il doit être abattu avec :
1. **Niyyah** (intention rituelle) — intention de l'abattre pour la consommation
2. **Tasmiyah** (invocation) — « Bismillah » (au nom d'Allah)

#### Réalité industrielle : CONDITIONS IMPOSSIBLES
La production industrielle du shellac implique :
- Extraction massive d'insectes
- Séchage à vapeur/chaleur (pas d'intention rituelle)
- Aucune invocation (tasmiyah)
- Filtration brute des restes

**Conclusion logique :** Même le madhab Maliki (classiquement le plus permissif) ne peut justifier le shellac industriel.

#### Position Maliki contemporaine :
- **Bin Bayyah** (Mauritanie) : Nuanced permissivism sur substances transformées, mais insiste sur conditions
- **ECFR (European Council)** : Permissivism partiel sur substances transformées

### Indices visuels :

- **Brillance excessive** sur fruits frais = probable shellac (comparé à fruits non cirés)
- **Couleur uniforme brillante** sur bonbons premium
- **Code E904** explicitement listé = certain

### Comment réagir si vous trouvez du shellac :

1. **Certification MUI/JAKIM visible ?** → Permissible (suivre votre madhab local)
2. **Région Indonésie/Malaisie/Égypte ?** → Permissible (confiance locale)
3. **Sans certification + région UK/Europe ?** → Mashbooh/Éviter
4. **Doute personnel ?** → Laver (fruits), ou choisir alternative sans shellac
5. **Usage cosmétique** → Permissible sans restriction supplémentaire

## 8. Verdict Naqiy final

### **Global Score : 45/100 — MASHBOOH (consommer par prudence)**

### Distinction usage INTERNE vs EXTERNE

#### **Usage INTERNE (ingestion)**
- **Verdict :** MASHBOOH — Pas de consensus
- **Recommandation :** Éviter par défaut SAUF:
  - Produit explicitement certifié halal par MUI/JAKIM/Dar al-Iftaa Egypt
  - Vous acceptez la position du madhab permettant (Maliki conditions, ou Hanafi moderne Taqi Usmani)
  - Région/pays où consommation locale acceptée
- **Principe de prudence :** « Délaisse ce qui t'inspire du doute pour ce qui ne t'en inspire pas » (Hadith)

#### **Usage EXTERNE (cosmétique, bois, isolation)**
- **Verdict :** PERMISSIBLE (tous les madhāhib)
- **Rationale :** Pas d'ingestion → enjeu insecte moins pertinent → transformation complète → usage externe jamais interdit
- **Produits :** Vernis ongles, rouges à lèvres, laques bois — SÛRS sans restriction

### Recommandations par profil

| Profil | Recommandation |
|---|---|
| **Hanafi stricte (Birmingham/DUA tradition)** | ÉVITER shellac interne. Permissible externe. |
| **Hanafi modérée (Mufti Taqi Usmani)** | Shellac halal. Peut consommer si pas de résidu éthanol >0.5%. |
| **Shafi'i** | Éviter (position classique: insects haram). Exception: sauterelle seule permise. |
| **Hanbali** | Éviter (prohibition majoritaire). Exception: nécessité médicale extrême. |
| **Maliki** | Classiquement permissible, mais conditions industrielles impossibles → pratiquement Mashbooh. |
| **Région Indonésie/Malaisie** | Suivre MUI/JAKIM (HALAL). |
| **Région Égypte** | Suivre Dar al-Iftaa Egypt (HALAL via istihalah). |
| **Région UK** | Suivre HMC/Darul Ifta Birmingham (HARAM). |
| **Afrique du Sud** | Suivre SANHA (HALAL) ou DUA (HARAM) par madhab. |

### Énoncé final Naqiy

> Le shellac (E904) est classé **MASHBOOH par défaut**. Aucun consensus institutionnel international. La majorité des autorités sunnites traditionnelles le prohibent (Hanafi classique, Shafi'i, Hanbali). Quelques autorités modernes/sud-asiatiques (MUI, JAKIM, Dar al-Iftaa Egypt, Mufti Taqi Usmani) le permettent via istihalah (transformation). **Recommandation Naqiy :** Évitez par défaut l'usage interne, acceptez l'usage externe. Pour usage interne, acceptez seulement sur certification explicite d'une autorité permissive ET cohérence avec le madhab du consommateur. Consultez votre Dar al-Iftaa local ou institution halal de confiance.
