## 1. IDENTIFICATION DES SUBSTANCES

### 1.1 Lactose (C₁₂H₂₂O₁₁)

| Paramètre | Valeur |
|---|---|
| Nom français | Lactose / Sucre lactique / Sucre du lait |
| Nom arabe | اللاكتوز / سكر اللبن |
| Code E | E966 (en certaines juridictions; NON universel) |
| Formule chimique | C₁₂H₂₂O₁₁ |
| Masse molaire | 342,30 g/mol |
| Numéro CAS | 63-42-3 |
| Concentration naturelle | 4-5% en poids dans lait bovin |

Le lactose est le disaccharide naturellement présent dans le lait de tous les mammifères. C'est le sucre du lait depuis 8000 ans de production laitière.

### 1.2 Lactosérum (Whey) — مصل اللبن

Le lactosérum est la portion liquide restante après coagulation du lait (fromage, yogurt).

**Composition :** Protéines (10-15% forme concentrée), lactose (40-50%), minéraux, eau

**Production mondiale :** 200 millions tonnes lait/an → 9 millions tonnes poudre whey/an (2024)

### 1.3 Types de Protéines de Whey

- **WPC (Concentré):** 25-89% protéine, 30-60% lactose
- **WPI (Isolat):** >90% protéine, <1% lactose  
- **WPH (Hydrolysat):** Peptides courts, absorption rapide

## 2. L'ENJEU CENTRAL: DÉPENDANCE À LA PRÉSURE

**Le lactosérum est un SOUS-PRODUIT de la coagulation du lait utilisant la présure (rennet).**

```
Lait → Présure (rennet) → Coagulation → Caillé (fromage) + Lactosérum
```

**Le lactosérum hérite du statut halal/haram de la présure utilisée.**

## 4. ANALYSE PAR MADHAB (4 Madhabs)

### HANAFI — HALAL (95%)

**Logique :** Rennet de tout animal halal (vivant ou mort) est pur car:
1. N'est pas partie de la chair (not lihm)
2. Sang ne circule pas (no dam)
3. Analogie avec lait dans le pis (qiyās al-laban fil-dāri')

**Savants contemporains :** Mufti Taqi Usmani, Mufti Ebrahim Desai, SeekersGuidance Hanafi

**Textes classiques :**
- al-Hidāya (al-Marghīnānī, 593 AH)
- Badā'iʿ al-Ṣanā'iʿ (al-Kāsānī, 587 AH)
- Radd al-Muḥtār (Ibn ʿĀbidīn, 1252 AH)

### MALIKI — HALAL CLASSIQUEMENT; MASHBOOH INDUSTRIELLEMENT (70%)

**Condition classique :** Niyyah (intention) + Tasmiyah (invocation "Bismillah")

**Réalité industrielle :** Production whey n'a JAMAIS ces conditions rituelles

**Résultat :** Même si rennet halal, conditions Maliki non satisfaites = mashbooh industriellement

### SHAFI'I — DÉPEND SOURCE (85%)

**Requis :** Rennet d'animal **zabiha** (properly slaughtered) car:
- Rennet dans stomach lining
- Stomach devient najis (impure) quand animal dies
- Contamination par contact (mulāqāh)

**Si halal-abattu :** HALAL
**Si maytah :** HARAM

**Textes classiques :** al-Umm (al-Shāfiʿī), al-Majmūʿ (al-Nawawī), Mughnī al-Muḥtāj (al-Sharbīnī)

**Modern Shafi'i :** Istihalah principle appliqué à sources microbiennes (non-animal)

### HANBALI — HARAM CLASSIQUEMENT; IBN TAYMIYYAH DISSIDENT (75%)

**Classique (Ibn Qudāma):** Maytah rennet = najis → fromage/whey haram

**Ibn Taymiyyah (minority):** Position Hanafi-like basée sur athar sahaba (Companions mangeaient fromage Zoroastrien)

**Modern Saudi (Lajnah Dā'imah):** Tolerance croissante de traces en produits transformés (istihlak principle)

**Textes classiques :** al-Mughnī (Ibn Qudāma, 620 AH), al-Inṣāf (al-Mardāwī, 885 AH)

## 5. CONSENSUS ET DIVERGENCE

### Zones de Consensus (IJMĀ')
- Lait animal halal = halal ✓ UNANIME
- Whey microbien = halal ✓ UNANIME
- Whey FPC = halal ✓ 95% CONSENSUS
- Whey végétal = halal ✓ UNANIME
- Whey halal-abattu = halal ✓ UNANIME

### Zones de Divergence (KHILĀF)
- **Whey de rennet animal maytah :**
  - Hanafi: HALAL (25% umma)
  - Shafi'i: HARAM (25% umma)
  - Maliki: HARAM (industriellement) (10% umma)
  - Hanbali: HARAM classique / HALAL Ibn Taymiyyah (10% umma)

**Impact marché :** 45% des fromages animaux commerciaux (AOC/AOP français) utilisent rennet maytah
